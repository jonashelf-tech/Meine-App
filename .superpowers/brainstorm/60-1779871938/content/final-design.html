Verhalten-Details

  • Chip-Tap → Zeitplan expandiert so dass die Stunde sichtbar wird (visStart oder visEnd wird angepasst)
  • + Button → wie bisher: eine Stunde früher/später hinzufügen
  • − Button → wie bisher: erste/letzte Stunde entfernen (nur wenn leer)
  • Leere Chip-Area → Bar bleibt sichtbar, Chip-Area ist leer (nur − und + bleiben)
  • Default → 8–18 Uhr (statt bisher 8–20)
