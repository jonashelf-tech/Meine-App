Wie kommt man in den Verwaltungs-Modus?

A

Stift-Icon → Verwaltungs-Modus

Neben den Chips gibt es ein kleines Stift-Icon. Antippen → alle Chips zeigen ein ✕ zum Löschen. Nochmal tippen → zurück zur Auswahl.

Normal
Kategorie
Nach Tipp auf Stift
Kategorie
Arbeit ✕ Privat ✕ Gesundheit ✕

Stift leuchtet lila wenn aktiv. Fertig = nochmal tippen.

B

Lang-Tipp auf Chip → Löschen

Normaler Tipp = Kategorie auswählen. Lang-Tipp auf einen Chip = dieser Chip bekommt ein ✕ und kann gelöscht werden. Einfacher, aber weniger sichtbar.

Normal
Kategorie
Nach Lang-Tipp auf "Arbeit"
Kategorie
Arbeit ✕

Nur der lang-getippte Chip zeigt ✕. Tipp woanders → zurück zum Normalzustand.
