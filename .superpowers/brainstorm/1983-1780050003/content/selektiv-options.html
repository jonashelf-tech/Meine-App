Selektive Aufmerksamkeit — 3 Vorschläge

Fokus auf einen Reiz, Ablenkungen aktiv rausfiltern. Was passt am besten?

A

Flanker Task

5 Pfeile in einer Reihe. Tippe Links oder Rechts — aber nur basierend auf dem mittleren Pfeil. Die äußeren zeigen manchmal dieselbe Richtung (leicht), manchmal die entgegengesetzte (schwer, Konflikt).

← ← → ← ←

Konflikt: außen ← aber Mitte → · Tippe RECHTS

← ← ← ← ←

Kongruent: alle ← · Tippe LINKS

Gemessen: RT bei Konflikt vs. kein Konflikt (Interferenz-Score). ADHS-relevant: Impulskontrolle bei Ablenkung.

B

Zielfarbe im Strom

Symbole blitzen einzeln auf — verschiedene Farben, verschiedene Formen. Nur tippen wenn das Symbol in der Zielfarbe erscheint (z.B. immer Lila). Alles andere ignorieren.

Ziel: Lila → nur beim dritten tippen

Zielfarbe: ● LILA

Gemessen: Treffer, Fehler, RT. Einfach zu verstehen, ähnlich CPT aber Filter statt Inhibition.

C

Odd-One-Out

Grid aus 9–16 Symbolen — eines weicht ab (andere Form oder andere Farbe). So schnell wie möglich das Ausreißer-Symbol tippen. Dann neues Grid.

Dreieck tippen — so schnell wie möglich

Gemessen: Zeit bis Treffer pro Grid, Fehler. Visuell interessant, aber ähnelt Zahlensuche in Struktur.
